# Channel Three Push System - TradingView Pine Script Indicator

## 📊 系统概述
这是一个基于"通道三段式推动"交易策略的TradingView Pine Script指标。系统自动识别急速行情、绘制通道、标记三段推动形态，并预警衰竭信号。

## 🚀 核心功能

### 1. 急速行情识别
- 自动检测连续大实体K线
- 多头/空头急速行情区分
- 实体大小阈值可配置

### 2. 通道绘制
- 线性回归计算趋势线
- 平行线绘制通道上下轨
- 自动标记通道起点和极值

### 3. 三段推动标记
- **第一段推动**：强劲上涨，回调浅
- **第二段推动**：动能衰减，高度<第一段80%
- **第三段推动**：衰竭过冲，突破后快速反转

### 4. 衰竭信号预警
- 突破后5根K线内反转验证
- 反转K线形态识别（吞没、Pin Bar）
- 自动警报和标记

### 5. 多重验证机制
- 通道斜率验证
- 摆动次数验证
- 实体大小衰减验证

## ⚙️ 参数配置

### 急速行情参数
- `rapidLen`: 检测窗口（默认: 10）
- `rapidMultiplier`: 实体倍数阈值（默认: 1.5）
- `minRapidBars`: 最少急速K线数（默认: 2）

### 通道参数
- `channelLookback`: 通道检测窗口（默认: 20）
- `minTouchPoints`: 最少接触点数（默认: 3）
- `channelSlopeThreshold`: 通道斜率阈值（默认: 0.6）

### 三段推动参数
- `push1MinHeight`: 第一段最小高度%（默认: 0.6）
- `push2MaxHeight`: 第二段最大高度%（默认: 0.8）
- `overshootThreshold`: 过冲阈值%（默认: 0.2）

### 验证参数
- `failureBars`: 衰竭验证K线数（默认: 5）
- `minSwingCount`: 最少摆动次数（默认: 2）
- `reversalConfirmation`: 反转确认K线数（默认: 1）

## 📈 使用方法

1. **添加到TradingView**
   - 复制`channel_three_push.pine`代码
   - 粘贴到Pine Editor
   - 点击"添加到图表"

2. **参数调整**
   - 根据市场调整参数
   - 建议从默认值开始
   - 观察信号准确性后微调

3. **最佳应用**
   - **品种**: XAUUSD黄金
   - **周期**: 1小时图
   - **参数**: 使用默认值开始

## 🎯 交易策略

### 做多机会
- **位置**: 第一段或第二段推动后的回调中
- **入场**: 通道下轨附近 + 反转信号
- **目标**: 通道上轨或前一段推动的高点
- **止损**: 通道下轨下方

### 做空机会
- **位置**: 第三段推动的衰竭点
- **入场**: 出现明确反转信号后
- **目标**: 通道起点（第一目标）、急速起点（第二目标）
- **止损**: 衰竭点上方

## ⚠️ 注意事项

1. **不要假设必然有三段**：市场可能只走两段就结束
2. **结合大周期背景**：日线通道 vs 小时线通道意义不同
3. **等待确认**：第三段推动必须确认衰竭
4. **风险管理**：通道交易本质是区间交易，止损要紧凑

## 📚 系统原理

基于7个验证模块：
1. 趋势结构识别（急速行情+通道）
2. 通道平衡验证（规律摆动）
3. 过冲衰竭识别（虚假突破）
4. 趋势线突破确认
5. 双重顶形态验证
6. 通道起点支撑测试
7. 交易区间形成与突破
